# 숫자 야구 게임

## 구현할 기능 목록
* 정답 생성. 서로 다른 수로 이루어진 숫자 3개 선택  
* 정답 입력 인터페이스 구현  
* 정답 입력값 유효성 체크. 실패시 에러메시지 출력  
* 정답 입력값과 정답을 비교  
* 비교결과 출력  
* 게임 재실행 입력 인터페이스 구현  
* 게임 재실행 입력값에 따라 동작 처리  

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
